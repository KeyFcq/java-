接口：
	接口使用关键字 interface 修饰 
	类实现接口用 implements 表示
	
	接口的变量是常量， 默认为用 public static final 修饰，可以简写为 int num1 =3； 可直接使用接口名称访问
	接口的方法必须是抽象方法，不能有具体的方法，默认为用 public abstract void 修饰，可以简写为 void method1();
	
	接口无构造器
	接口通过实现类来实例化 (接口多态)
	
	多态：有继承或者实现关系；有方法重写；有父(类/接口)指向引用(子/实现)类对象
	Object 是所有类的超类，一个类如果没有父类，默认继承自Object
	
	接口的实现类：
		重写接口中所有抽象方法或者 实现类是抽象类
