继承变量使用：
1、首先找方法里面的定义的变量
2、方法里面没有就在本类里面找
3、本类没有就在父类里面找

若方法里面，本类里面，和父类里面都有，则本类 用this ，父类用super

子类的构造器，默认会调用 父类的无参构造器(若指定了有参构造器，则需要手动指定无参构造器)

子类的方法和变量一样，首先在子类里面找，若没有则去父类找，（extends 只找父类，不找父类的父类 ）

继承的注意事项：
		java 类只支持单继承，不支持多继承
		java 类支持多层继承

方法的重写：
	若子类和父类方法相同，子类中方法还需要一些特有的功能，则可以在子类方法里面写 super.方法名 + 子类特有的功能
	
	注意：若担心重新的方法名写的不对，可以在方法前面加上 @override	
				申明为 static 、private、 final 的方法不能被重写
				构造方法不能重写
				重写：返回值、函数名、参数(包括参数个数,对应位置参数类型)  三个元素必须相等
				子类重新方法的修饰符必须和父类相同或者访问范围比父类大
				
				关于修饰符的重写和访问范围：
					 修饰符				重写                          访问范围
					private   不能被重写										本类
					default   同一个包中的子类可以重写      同一个包
					protectde 都可以重写										同一个包，或者不同包但是必须是子类
					public    都可以重写										所有范围
